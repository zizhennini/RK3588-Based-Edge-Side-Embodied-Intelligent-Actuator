# scripts — 工具脚本

## 启动脚本

| 脚本 | 用途 |
|------|------|
| `teleop.sh` | 主从遥操作示教 |
| `record.sh` | 示教数据录制 |
| `run_vla.sh` | VLA 自主抓取 |

## 标定脚本

| 脚本 | 用途 |
|------|------|
| `calibrate_camera.py` | 棋盘格相机内参标定 |
| `calibrate_handeye.py` | Tsai-Lenz 手眼标定 |

## 系统工具

| 脚本 | 用途 |
|------|------|
| `monitor.py` | RK3588 NPU/CPU/内存实时监控 |
| `rkeia.service` | systemd 开机自启服务 |
